## 📋 Описание

Репозиторий содержит материалы и задания по дисциплине **«Базы данных»**.

В рамках курса выполнялись задания по работе с **реляционными и нереляционными базами данных**, написанию **SQL-запросов** (MySQL), а также разработке **приложений на C# (Windows Forms)** для взаимодействия с базой данных.

---

## 📚 Темы курса

* основы реляционных и нереляционных баз данных;
* язык запросов **SQL** (MySQL);
* создание и изменение таблиц;
* работа с данными (`SELECT`, `INSERT`, `UPDATE`, `DELETE`);
* связи между таблицами;
* разработка приложений для работы с базами данных;
* и др.
